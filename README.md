# FoodHub
FoodieHub is a modern, fully responsive restaurant web application built with React, Vite, and TailwindCSS. It offers a smooth and dynamic user experience with interactive UI elements, category-based menu filtering, a working cart system (using local storage), and a clean, visually appealing design.
